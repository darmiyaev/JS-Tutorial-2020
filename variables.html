Why learn JS?

JS makes it possible to make your website dynamic.
This means that you can make your website interactive with JS.

What can JS do?

This is a question that I asked for many programming languages, and here is a clear list:

  1. Make websites interactive (buttons, games etc)
  2. Make games (on websites)
  3. Make desktop apps (electron)
  4. It can be used with library's/frameworks etc (react JS, node JS)
