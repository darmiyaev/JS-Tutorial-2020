Variables

So what are variables?
Variables hold values.
Values are just data or information.
Variables are used a lot in coding and are simple.
Here is how you use variables in Javascript:
var x = 0;
Ok now let's break this down.
var is telling the computer that we are creating a variable.
x is the name of the variable (information)
= is assigning a value. (= is giving a value to the variable, x) 0 is the value of the variable.
; is ending the statement, statement is just the line of code.
